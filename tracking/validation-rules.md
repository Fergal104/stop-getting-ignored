# Validation Rules

## Day 3

- Goal: first posts published and first landing-page visits arriving.
- If visits are zero, the posting volume is too low or links are missing.

## Day 7

- Goal: 50 landing visitors and at least 5 free downloads.
- If visitors are below 50, improve hooks and posting consistency.
- If visitors are above 50 but downloads are below 5, improve hero copy and CTA prominence.

## Day 14

- Goal: first sale.
- If downloads exist but sales do not, improve the free-to-paid bridge and Gumroad page.
- If checkout clicks exist but sales do not, test price framing and stronger product bullets.
- If no sale exists by day 14, keep the system but change the niche angle first.

## Day 21

- Goal: 3 sales or a clear positive conversion trend.
- If there is still no traction, rebuild the front-end angle around one narrower niche, such as web designers or email marketers.
