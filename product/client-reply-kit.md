# Stop Getting Ignored

Use this kit as a starting library, not a substitute for thinking. Replace the bracketed placeholders, keep the message short, and anchor each opener to something visible.

## Quick Rules

- Open with an observation, not a compliment.
- Point to one visible problem or missed opportunity.
- Make the ask small.
- Do not stack multiple asks into one message.
- Follow up with new context, not guilt.

## Discovery Call Script

### Opening

- What prompted you to look at this now instead of later?
- Where are you seeing the biggest drop-off: replies, booked calls, proposals, or closes?
- What have you already tried that did not fix it?

### Problem Diagnosis

- What does your current outreach or conversion path look like today?
- Where do prospects seem interested but stop moving?
- Which offer or service matters most to sell first?

### Proof and Constraints

- What proof do you already have that is underused?
- How quickly do you need this fixed?
- Are there any messaging constraints, compliance issues, or approval steps?

### Close

- If I show you a focused fix instead of a big rebuild, would that be the preferred route?
- Who needs to approve the work?
- What would make this an obvious yes for you?

## Objection Handling Sheet

| Objection | Reply |
|---|---|
| We already tried outreach. | That usually means the angle or framing was weak, not that outbound itself is dead. |
| We do not have time right now. | That is why a small fix is the first step. This is built to reduce decision load, not add a long project. |
| We can write this ourselves. | You probably can. The value here is getting to a cleaner version faster with less trial and error. |
| We need more leads, not better messaging. | Better messaging usually improves the value of every lead source you already have. |
| Can you guarantee results? | No credible operator should. The work is about improving clarity, trust, and reply odds. |
| Why not just ask for a call? | Lower-friction asks tend to earn more engagement early, especially with colder prospects. |
