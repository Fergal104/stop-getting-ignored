# Automation Logic

## Core Production Loop

One core insight becomes a small asset cluster:

1. One short post for X
2. One conversational variation for Threads
3. One lead-magnet teaser line
4. One paid-kit bridge sentence

## Weekly Input

Start with one real observation from:
- a weak outreach message
- a weak landing page
- a bad CTA
- a poor follow-up
- a visible proof problem

## Conversion Logic

- Public content builds trust around one narrow pain.
- The free pack solves one small problem immediately.
- The paid kit expands that trust into a reusable system.
- Strong posts are recycled into future lead magnets or niche editions.

## Trigger Rules

- If a post gets above-average engagement, rewrite it into a longer Threads variant and email angle.
- If a post gets clicks but no downloads, sharpen the landing-page hook.
- If downloads happen without sales, add a stronger bridge email and a clearer Gumroad bullet list.
- If buyers reply with a niche question, log it and build a niche-specific edition later.

## Low-Maintenance Rule

Do not create new products until one product has:
- 100 visitors
- at least 1 sale
- at least 3 posts that consistently outperform the rest
